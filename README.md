# 🛒 Tryvion E-Commerce Website

A full-stack E-Commerce web application built using **Django** and **Python**, designed to provide a seamless online shopping experience. The platform allows users to browse products, manage their cart, place orders, and manage their accounts, while administrators can efficiently manage products, categories, users, and orders through the Django Admin Panel.

## ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git
cd YOUR_REPOSITORY_NAME
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

### Run Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```text
http://127.0.0.1:8000/
```
